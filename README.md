# hterm

A simple terminal server for the web. Inspired by [ttyd](https://github.com/tsl0922/ttyd).

<img src=".github/images/hterm.png" alt="hterm screenshot" />

## Install

```bash
curl -s https://i.jpillora.com/Rishang/hterm! | bash
```

## Usage

```bash
hterm -W -p 7681
```

### Usage help

```bash
hterm --help
```
